# Linux-commands-
 
## File and Directory Commands

| Command | What it does | Example |
|---------|--------------|---------|
| `ls` | List files and directories | `ls` |
| `ls -l` | List contents of current directory (long format) | `ls -l` |
| `ls -al` | List all files including hidden ones | `ls -al` |
| `cd` | Change directories | `cd /home/user` |
| `find ~ -name` | Search for files | `find ~ -name file.txt` |
| `rm` | Remove file or directory | `rm file.txt` |
| `cat` | Display the contents of a file | `cat file.txt` |
| `pwd` | Print the working directory | `pwd` |
| `cd ..` | Go back one level | `cd ..` |
| `touch` | Create an empty file | `touch file.txt` |
| `mv` | Move or rename a file | `mv old.txt new.txt` |
| `less` | View file content page by page | `less file.txt` |
| `head` | Show the first lines of a file | `head -n 10 file.txt` |
| `tail` | Show the last lines of a file | `tail -n 10 file.txt` |

## Investigating the System

| Command | What it does | Example |
|---------|--------------|---------|
| `whoami` | Print current username | `whoami` |
| `uname -a` | Show detailed info about the OS/kernel | `uname -a` |
| `uname` | Print OS name | `uname` |
| `df -h` | Check disk space and storage usage | `df -h` |
| `top` | Display running processes | `top` |
| `cat /etc/os-release` | Show details about the Linux distribution | `cat /etc/os-release` |
| `cd /etc` | Navigate to config/info files directory | `cd /etc` |
| `id` | Show user and group info | `id` |
